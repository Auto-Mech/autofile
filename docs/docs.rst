Documentation
=============

Overview
~~~~~~~~

The filesystem schema used by Auto Mech is implemented in
:doc:`autofile.fs <./submodule_fs>`.
This module will be of most interest to users of Auto Mech.

The other submodules provide the framework by which this schema is implemented
and can in principle be used to create other filesystems.
These are:
:doc:`autofile.system <./submodule_system>`,
:doc:`autofile.file <./submodule_file>`, and
:doc:`autofile.info <./submodule_info>`.
